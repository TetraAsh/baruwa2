
============
Requirements
============

Baruwa 2.0 is built on best of breed open source packages.

Baruwa Requirements
===================

	+ Python >= 2.6 < 3.0
	+ Pylons
	+ Celery
	+ Psycopg
	+ PostgreSQL
	+ GeoIP-Python
	+ iPy
	+ A web server that supports WGSI
	+ Reportlab
	+ Lxml
	+ SQLAlchemy
	+ Memcached
	+ Sphinx
	+ M2Crypto
	+ repoze.what
	+ repoze.who
	+ marrow.mailer
	+ pyrad
	+ python-ldap

MailScanner Requirements
========================

	+ MailScanner >= 4.80.10-1
	+ DBI
	+ DBD-Pg
	+ DBD-MySQL
	+ DBD-SQLite
